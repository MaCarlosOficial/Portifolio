Banco de Dados Não Relacional - NoSQL

O 1º banco de dados NoSQL surgiu no final da década de 1990 com o nome Strozzi NoSQL, nome dado pelo seu criador Carlo Strozzi. Os dados eram armazenados em arquivos no formato ASCII e cada tupla era representada por uma linha com os campos separados por tabulações. Em 2009 surgiu um novo conceito para os bancos NoSQL, em um reunião em São Francisco US, organizada por Johan Oskarsson. Começou então a surgirem novos projetos como o BigTable (Google) e o Dynamo (Amazon).

Redes Neurais

As Redes Neurais Artificiais ou RNA, são um subconjunto de Machine Learning. Seu nome e estrutura, foram inspirados no cérebro humano, simulando a maneira como os neurônios humanos biológicos enviam sinais uns para os outros, com o objetivo de adquirirem conhecimento através da experiência.

Databricks

A Databricks é uma empresa americana de software fundada pelos criadores do Apache Spark. A Databricks desenvolve uma plataforma baseada na web para trabalhar com o Spark, que fornece gerenciamento automatizado de cluster e notebooks no estilo IPython (wikipedia)

Spark

O Apache Spark é um mecanismo de análise unificado para processamento de dados em grande escala com módulos integrados para SQL, streaming, machine learning e processamento de gráficos. (Google Cloud)

ETL

ETL é a sigla para o processo de extrair, transformar e carregar. É uma forma tradicionalmente aceita para que as organizações combinem dados de vários sistemas em um único banco de dados, repositório de dados, armazenamento de dados ou data lake. O ETL pode ser usado para armazenar dados legados, ou, o que é mais comum, agregar dados para analisar e impulsionar as decisões de negócios. (Google Cloud)
